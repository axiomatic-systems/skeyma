Create a new key, value generated by the server, nothing specified:
------
POST http://localhost:3000/keys?kek=00112233445566778899aabbccddeeff
Body: None

Create a new key, value generated by the server, kid specified:
------
POST http://localhost:3000/keys?kek=00112233445566778899aabbccddeeff
Body: { "kid": "a0a1a2a3a4a5a6a7a8a9aaabacadaeaf" }

Create a new key, value generated by the server, kekId specified:
------
POST http://localhost:3000/keys?kek=00112233445566778899aabbccddeeff
Body: { "kekId": "@myvalue" }

Create a new key, value generated by the server, info specified:
------
POST http://localhost:3000/keys?kek=00112233445566778899aabbccddeeff
Body: { "info": "some comment" }

Create a new key, value generated by the client, unwrapped, kid unspecified:
------
POST http://localhost:3000/keys?kek=00112233445566778899aabbccddeeff
Body: { "k": "000102030405060708090a0b0c0d0e0f" }

Create a new key, value generated by the client, wrapped:
------
POST http://localhost:3000/keys
Body: { "ek": "40d529934420695da092f5f57100d561e7679aa50f779d5a", "kekId"="@blabla" }
 
Get all keys (encrypted)
------------
GET http://localhost:3000/keys

Get all keys (decrypted)
------------
GET http://localhost:3000/keys?kek=00112233445566778899aabbccddeeff

Get one key (encrypted)
-----------
GET http://localhost:3000/keys/a0a1a2a3a4a5a6a7a8a9aaabacadaeaf

Get one key value (encrypted)
----------------
GET http://localhost:3000/keys/a0a1a2a3a4a5a6a7a8a9aaabacadaeaf/value

Get one key (decrypted)
-----------
GET http://localhost:3000/keys/a0a1a2a3a4a5a6a7a8a9aaabacadaeaf?kek=00112233445566778899aabbccddeeff

Get one key value (decrypted)
----------------
GET http://localhost:3000/keys/a0a1a2a3a4a5a6a7a8a9aaabacadaeaf/value?kek=00112233445566778899aabbccddeeff

Change a key value
------------------
PUT http://localhost:3000/keys/a0a1a2a3a4a5a6a7a8a9aaabacadaeaf
